## **Bash with Loop and Function**

The following code is an example of a Bash script that uses a loop and a function.

```bash
#!/bin/bash

# A function that prints the given message
printMessage() {
  echo $1
}

# The while loop
while true; do
  printMessage "Hello World!"
  sleep 1
done
```
