

**Solution:**

Let $u = \tan x \Rightarrow du = \sec^2 x\ dx$

$$\int \tan^4 x \bigg(1 + \tan^2 x\bigg)\ d\tan x = \int \tan^4 x \ d\tan x + \int \tan^6 x \ d\tan x$$

$$\begin{align*}
&= \int u^4\ du + \int u^6\ du \\
&= \frac{u^5}{5} + \frac{u^7}{7} + C \\
&= \frac{\tan^5 x}{5} + \frac{\tan^7 x}{7} + C
\end{align*}$$
